WEASLEYS' WIZARD WHEEZES

Готовая папка для VS Code.

Как открыть:
1. Распакуйте архив.
2. В VS Code выберите File / Open Folder.
3. Откройте папку weasleys-vscode-ready.
4. Запустите index.html через Live Server или откройте файл в браузере.

Страницы:
- index.html — главная
- product.html — карточка товара
- cart.html — корзина
- payment.html — оплата
- tracking.html — отслеживание заказа

Картинки лежат в папке assets. Названия файлов не изменялись.
